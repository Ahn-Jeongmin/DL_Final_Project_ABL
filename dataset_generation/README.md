
# Data Generation

This folder contains the necessary scripts and configurations to generate the custom dataset for the DL_Final_Project_ABL. The process involves creating the initial high-quality data required for the Supervised Fine-Tuning (SFT) phase.

## How to Run

Execute the primary script to initiate the data creation process:

```bash
python entity(step1).py
python qa(step2).py
````

Please ensure all dependencies are properly installed before running.

## Pre-Generated Dataset Download

A complete, pre-generated dataset is available for direct download. Using this will allow you to skip the data generation step and proceed straight to model training.
